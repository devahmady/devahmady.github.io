Script ON-OFF Ether Mikrotik Scheduler

From Input

Fill in according to Interface and Ip

[ Senin ]
[ Selasa ]
[ Rabu ]
[ Kamis ]
[ Jum'at ]
[ Sabtu ]
[ Minggu ]

please leave a comment if an error occurs in the script

Output Script

Paste the script into the mikrotik terminal

/system scheduler add name=start-senin interval="1d 00:00:00" start-time="{{start1}}" on-event="/interface disable {{et1}}" comment=1.start-senin
/system scheduler add name=on-senin interval="1d 00:00:00" start-time="{{on1}}" on-event="/interface enable {{et1}}" comment=1.on-senin
